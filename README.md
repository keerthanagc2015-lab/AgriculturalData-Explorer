# 🌾 AgriScope – Agricultural Analytics Dashboard

> A Power BI analytics solution for exploring crop production,
> yield trends, and cultivation patterns across Indian states.

## 📌 Overview
AgriScope is an end-to-end agricultural data analytics dashboard
built to uncover insights from Indian crop data. It enables
state-wise and year-wise exploration of production performance,
cultivation area, and yield efficiency through interactive visuals.
Exploratory Data Analysis (EDA) was performed using Python to
validate, profile, and prepare the dataset before visualization.

## 🚀 Features
- 🐍 EDA using Python — data profiling, null handling, outlier detection
- 📊 Dynamic Power BI dashboards with slicers and drill-throughs
- 🗺️ State-wise & year-wise crop production analysis
- 📈 Year-over-year agricultural growth tracking
- 🔗 Correlation analysis between cultivation area and crop yield
- 🧹 Data cleaning and transformation via Power Query

## 🛠️ Tech Stack
| Tool         | Purpose                              |
|--------------|--------------------------------------|
| Python       | EDA, data profiling & preprocessing  |
| SQL          | Data extraction & querying           |
| Power Query  | Data transformation & cleaning       |
| DAX          | Calculated measures & KPIs           |
| Power BI     | Dashboard & visualization layer      |

## 🐍 Python EDA Highlights
- Null value analysis and imputation strategy
- Distribution plots for crop production and yield
- State-wise and crop-wise aggregation summaries
- Correlation heatmap — cultivation area vs. production
- Outlier detection using IQR method
